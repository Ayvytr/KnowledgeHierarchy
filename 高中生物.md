# 高中生物

## 从生物圈到细胞

生物分类：单细胞生物，多细胞生物，非细胞生物（病毒没有细胞结构，不能独立生存）



### 生命系统结构层次

#### 结构层次

细胞：最基本的生命系统，是生物体结构和功能的基本单位

组织：形态相似，功能相同的细胞联合在一起构成的细胞群

器官：不同的组织按照一定次序结合在一起构成的，能行使一定功能的结构单位

系统：能够共同完成一种或几种生理功能的多个器官按照一定次序组合在一起形成的整体

个体：由各器官或系统协调配合共同完成复杂生命活动的生物体。单细胞生物由一个细胞构成

种群：在一定自然区域内，同种生物的所有个体的总和，是生物繁殖的基本单位，也是进化的基本单位

群落：同一时间内聚集在同一区域中各种生物的集合

生态系统：生物群落与它的无机环境相互作用而形成的统一整体

生物圈：地球上全部生物及其无机环境共同组成，是最大的生态系统



#### 特例

* 植物没有系统这一层次
* 单细胞生物既可以看做个体层次，又可以看做细胞层次，没有组织、器官、系统这三层次
* 病毒没有细胞结构，不属于生命系统，但其生命活动依赖生命系统且属于生物
* 细胞是最小的结构层次，组成细胞的元素、组成细胞的蛋白质、核酸等化合物和细胞膜、细胞器等结构都不能独立表现生命特征，不属于生命系统的结构层次

### 原核细胞和真核细胞

#### 组成

原核细胞：细胞壁、细胞膜、拟核、核糖体

真核细胞

* 植物细胞：细胞壁、液泡、细胞膜、叶绿体，细胞核，细胞膜

* 动物细胞：线粒体、内质网、核糖体、高尔基体，细胞核，细胞膜

#### 常见原核生物和真核生物的判断

* 菌类的判断

  菌字前面有“杆”、“球”、“螺旋”、“弧”、的都是细菌，为原核生物；酵母菌、霉菌、食用菌是真核生物

* 藻类的判断

  蓝藻是原核生物，其他藻类是真核生物



### 细胞的多样性和统一性



#### 统一性

* 基本结构：真、原核细胞都有细胞膜和细胞质，都含有与遗传有关的DNA分子
* 化学组成：不同细胞具有基本相同的元素组成和化合物种类
* 细胞来源：同一个生物个体的不同细胞一般都最终来自同一个细胞（如受精卵）的分裂

#### 以细胞多样性和统一性为基础的生物分类系统

生物

* 非细胞结构生物
  * 类病毒：仅由RNA分子组成
  * 朊病毒：有蛋白质组成，如疯牛病病原体
  * 由蛋白质和核算（DNA或RNA）组成的病毒：如$T_2$噬菌体，烟草花叶病毒等
* 细胞结构生物
  * 原核生物
    * 细菌：球菌、杆菌、螺旋菌等
    * 蓝藻：念珠藻、颤藻、发菜等
    * 放线菌、支原体、衣原体、立克次氏体
  * 真核生物
    * 真菌：酵母菌、青霉菌、蘑菇等
    * 植物：低等、高等植物
    * 动物：低等、高等动物



### 细胞学说

#### 内容

* 细胞是一个有机体，一切动植物都由细胞发育而来，并由细胞核细胞产物所构成
* 细胞是一个相对独立的单位，既有它自己的生命，又对与其他细胞共同组成整体的生命起作用
* 新细胞可以从老细胞中产生



# 组成细胞的分子

## 细胞中的元素和化合物

### 生物界与非生物界的统一性和差异性

统一性：组成细胞的化学元素在无机自然界都能找到

差异性：组成生物体的化学元素在生物体内和自然界中的相对含量相差很大



### 细胞中元素的种类与含量

大量元素：C H O N P S K Ca Mg 

微量元素：Fe Mn B Zn Cu Mo

记法：

* 大量元素：她请杨丹留人盖美家

* 微量元素：Fe Mn B Zn Mo Cu 铁猛碰新木桶

组成细胞的化学元素有20多种，按含量可分为大量元素和微量元素，按作用大小可分为基本元素，最基本元素，和主要元素

最基本元素：C

基本元素：C H O N 

主要元素：C H O N P S



大量元素和微量元素划分依据是含量，不是生理作用的大小。微量元素含量虽少，但是作用不可替代。

细胞的组成元素中，原子个数最多的元素是H；细胞鲜重中含量最多的元素是O，其次是C,H,N；细胞干重中含量最多的元素是C，其次是O,N,H



### 组成细胞的化合物

化合物：

* 无机物
  * 水 85%~90%
  * 无机盐 1%~1.5%
* 有机物
  * 糖类 （和核酸共1%~1.5%
  * 核酸
  * 蛋白质 7%~10%
  * 脂质 1%~2%



### 实验：生物组织中糖类、脂肪和蛋白质的检测

#### 原理

还原糖+斐林试剂$\xrightarrow[水浴加热]{50-65°C}$砖红色沉淀（无色→浅蓝色→棕色→砖红色）

淀粉+碘→蓝色

脂肪+苏丹III染液$\xrightarrow{镜检}$橘黄色

脂肪+苏丹IV染液$\xrightarrow{镜检}$红色

蛋白质+双缩脲试剂→紫色反应



还原糖反应中，需水浴加热，不能用酒精灯直接加热。脂肪检测中，需用50%的酒精洗去浮色，再借助显微镜观察。



## 生命活动的主要承担者——蛋白质

## 氨基酸的结构

### 结构通式

![](media/氨基酸.png)

R基：决定氨基酸种类

氨基$H_2N$：与相邻的COOH脱水缩合形成肽键

羧（suō）基：与相邻的$NH_2$脱水缩合形成肽键



#### 特点

* 至少含有一个氨基和一个羧基，R基中也可能含有氨基或羧基
* 都有一个氨基和一个羧基连在同一个碳原子上
* 不同氨基酸的R基不同，R基决定氨基酸的种类和性质



### 氨基酸的种类

组成蛋白质的氨基酸约有20多种，包括必须氨基酸和非必须氨基酸。包括赖氨酸、色氨酸、苯丙氨酸、亮氨酸、异亮氨酸、算算、甲硫氨酸和缬（xié）氨酸。婴儿比成人多一种组氨酸。

必须氨基酸：人体不能合成，必须从外接直接获取。

非必须氨基酸：可以通过其他化合物转化而来。



### 氨基酸脱水缩合形成肽链

![](media/氨基酸脱水缩合.jpg)

脱水缩合：一个氨基酸的羧基（-COOH）和另一个氨基酸的氨基（-$NH_2$）相连接，同时脱去一分子水，这种结合方式叫脱水缩合，脱下的水中，O来自-COOH,H来自-COOH和-$NH_2$,形成的化学键叫做肽键。肽键可以表示为-CO-NH-或-NH-CO-。

肽链：两个氨基酸分子脱水缩合形成的化合物叫二肽，多个氨基酸分子脱水缩合形成的含有多个肽键的化合物叫多肽，多肽一般呈链状结构，叫肽链。

蛋白质形成：一条或多条肽链经盘曲、折叠形成具有一定空间结构，行使一定功能的蛋白质

蛋白质的结构层次：C、H、O、N等元素$\xrightarrow{组成}$氨基酸$\xrightarrow{脱水缩合}$肽链$\xrightarrow{盘曲折叠}$蛋白质

### 蛋白质结构多样性的原因

* 氨基酸种类不同，构成的肽链不同
* 氨基酸的数目不同，构成的肽链不同
* 排列顺序不同，构成的肽链不同
* 盘曲折叠方式及形成的空间结构不同，构成的肽链不同

### 蛋白质的功能

结构蛋白：构成细胞核生物体结构的重要物质，比如细胞膜上的固着蛋白，组成核糖体、中心体、染色体的蛋白质。

功能蛋白：

* 调节作用：调节细胞代谢和生物体的生命活动。如胰岛素、生长激素等。
* 催化作用：催化细胞中各种化学反应的顺利进行，如绝大多数的酶。
* 运输作用：如细胞膜上的蛋白质载体、红细胞中的血红蛋白等有运输作用
* 免疫作用：能清除侵入体内的抗原物质。如抗体

### 氨基酸形成多肽中的相关计算（直链肽）

#### 多肽种类的计算——排列组合法

* 假若有A、B、C三种氨基酸，在每种氨基酸数目无限的情况下，可形成肽类化合物的种类：

  三肽：$3^3=27$种

  二肽：$3^2=9$种

* 假若有A、B、C三种氨基酸，在每种氨基酸只有一个的情况下，可形成肽类化合物的种类：

  三肽：$3×2×1=6$种

  二肽：$3×2=6$种

#### 蛋白质分子质量、氨基酸数、肽链数、肽键数和脱去水分子数的关系

肽键数=脱去水分子数=氨基酸数-肽链数

蛋白质分子质量=氨基酸数目×氨基酸平均相对分子质量-脱去水分子数×18

| 肽链数目 | 氨基酸数 | 氨基酸平均分子质量 | 肽链数目 | 多肽链分子量 | 氨基数目 | 羧基数目 |
| -------- | -------- | ------------------ | -------- | ------------ | -------- | -------- |
| 1条      | m        | a                  | m-1      | am-18(m-1)   | 至少1个  | 至少1个  |
| n条      | m        | a                  | m-n      | am-18(m-n)   | 至少n个  | 至少n个  |

#### 蛋白质中游离氨基或羧基数的计算

* 至少含有的游离氨基或羧基数=肽链数
* 游离氨基或羧基数=肽链数+R基中的氨基或羧基数

#### 蛋白质中含有N、O原子数的计算

N原子数=肽键数+肽链数+R基上的N原子数=各氨基酸中N原子总数

O原子数=肽键数+2×肽链数+R基上的O原子数=各氨基酸中O原子总数-脱去水分子数





## 遗传信息的携带者——核酸

### 观察DNA和RNA在细胞中的分布

实验原理：甲基绿和吡罗红试剂对DNA和RNA的亲和力不同

甲基绿+DNA→呈现绿色

吡罗红+RNA→呈现红色

实验现象：绿色主要集中在细胞中央，红色散布在绿色周围，分布较广，不集中。

结论：

* DNA主要分布在细胞核中

* RNA主要分布在细胞质中

  叶绿体、线粒体中也含有少量DNA和RNA



### 核酸的结构和功能

### 基本组成单位——核苷酸

种类：分脱氧（核糖）核苷酸和核糖核苷酸



结构：

含氮碱基A T G C和五碳糖脱氧核糖参与脱氧核苷酸构成

含氮碱基A U G C和五碳糖核糖参与核糖核苷酸构成



核酸的组成

核苷酸-脱氧核苷酸→脱氧核苷酸链$\xrightarrow{2条}$DNA-核酸

核苷酸-核糖核苷酸→核糖核苷酸链$\xrightarrow{1条}$RNA-核酸



核酸的功能：细胞内携带遗传信息的物质，在生物体的遗传、变异和蛋白质的生物合成物中具有极其重要的作用。

## 细胞中的糖类和蛋白质

### 糖类

#### 按水解后产生单糖的多少分类

* 单糖：不能水解的糖
  * 五碳糖：核糖，RNA的组成成分；脱氧核糖：DNA的组成成分。动植物细胞中都有
  * 六碳糖：葡萄糖：动植物细胞中都有分布，是细胞生命活动所需的主要生命物质；果糖，植物细胞中特有的单糖；半乳糖，动物细胞中特有的单糖。
* 二糖：水解后产生两分子单糖的糖
  * 乳糖：分布在动物的乳汁中，有半乳糖和葡萄糖缩合而成
  * 麦芽糖：分布在植物细胞中，由两分子葡萄糖缩合而成，在发芽的谷粒中含量较多
  * 蔗糖：分布在植物细胞中，由果糖和葡萄套缩合而成，在甜菜、甘蔗等植物中含量较多
* 多糖：水解后生成多个单糖的糖
  * 淀粉：分布在植物细胞中，是植物细胞中的储能物质
  * 纤维素：分布在植物细胞中，是植物细胞壁的主要组成成分
  * 汤圆：动物细胞中的储能物质。肝糖原主要分布在动物肝脏中，可调节血糖；肌糖原分布在动物肌肉组织中。

动物细胞特有糖类：糖原、乳糖、半乳糖

植物细胞特有糖类：蔗糖、淀粉、麦芽糖、纤维素

动植物细胞都有糖类：核糖、脱氧核糖、葡萄糖



#### 按还原性有无分类

还原糖：可与斐林试剂反应生成砖红色沉淀。有葡萄糖、果糖、半乳糖、乳糖、麦芽糖等。

非还原糖：不能与斐林试剂反应。有蔗糖、淀粉、纤维素等。



#### 糖类的功能

* 糖类是生物体维持生命活动的主要能量来源，生物体所需能量主要来自于糖类物质的氧化分解
* 糖类是细胞结构的组成成分。如纤维素是植物细胞壁的主要成分，核糖和脱氧核糖分别是RNA和DNA的组成成分
* 糖类可与蛋白质等其他物质结合形成复杂化合物，参与生命多种活动。如糖类与蛋白质结合形成糖蛋白，糖类与脂质形成糖脂



### 细胞中的脂质

#### 脂肪

作用：主要的储能物质；皮下脂肪层起到保温作用；内脏器官周围的脂肪具有缓冲和减压作用，保护内脏器官。

分布：植物种子、果实、动物的脂肪组织中

#### 磷脂

作用：构成细胞膜及多种细胞器膜的重要成分

分布：动物脑、卵细胞、肝脏及大豆种子中

#### 固醇

胆固醇：构成动物细胞膜的重要成分，在人体内参与血液中的脂质运输

性激素：促进人和动物生殖器官的发育及生殖器宝的形成

维生素D：促进人和动物肠道队钙和磷的吸收



### 糖类和脂质的比较

|          | 糖类                                             | 脂质             |
| -------- | ------------------------------------------------ | ---------------- |
| 元素     | C H O                                            | C H O (N P)      |
| 种类     | 单糖、二糖、多糖                                 | 脂肪、磷脂、固醇 |
| 合成部位 | 淀粉——叶绿体；纤维素——高尔基体；糖原——肝脏、肌肉 | 内质网           |

联系：糖类$\leftrightarrows$脂质



### 细胞中的能源物质

能源物质：细胞中，糖类、脂肪、蛋白质都是有机化合物，其中贮存有大量化学能。当其被氧化分解时，这些化学能就释放出来，供生命活动利用。

主要能源物质：糖类。糖类分解供能约占总能量70%以上。

储能物质：脂肪。动物细胞的糖原和植物细胞的淀粉也是重要的储能物质。

直接能源物质：ATP。生命活动所需能量由ATP水解提供，贮能物质所含能量只有转移到ATP中才能用于各项生命活动。

三大能源物质的功能顺序：先是糖类氧化供能，当糖类供能不足时，依次由脂肪、蛋白质供能；蛋白质除了在正常代谢中提供部分能量外，一般不提供。当需要由蛋白质大量供能时，说明生物体已病重或生命接近终结。



### 生物大分子以肽链为骨架

生物大分子是多具体，有单体（基本单位）脱水缩聚而成

| 单体（基本单位） | 多聚体（生物大分子） |
| ---------------- | -------------------- |
| 葡萄糖           | 多糖                 |
| 氨基酸           | 蛋白质               |
| 核苷酸           | 核酸                 |

每一个单体都以若干个相连的碳原子构成的碳链为基本骨架，由许多单体连城多聚体。
